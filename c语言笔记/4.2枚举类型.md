# 枚举类型

变量的取值范围是限制在一个有限的范围内

格式：

```c
enum 枚举名 {
	枚举值表;
}

enum weekday {
  sun, mon, tue, wed, thu, fri, sat
};
// 每个值对应一个序号，默认从0开始依次递增
// 当然可以自己定义，后续的从此值继续递增
enum right {
  true=10, false
}
```

定义方式：三种，和共用体他们一样

```c
// 第一种
enum right {
  true, false
};
enum right a,b,c;

// 第二种
enum right {
  true, false
}a,b,c;

// 第三种
enum {
  true,
}a, b, c;

```

注意：

（1）不能对枚举类型再进行赋值

（2）只能把枚举值赋予枚举变量，不能把元素的数值直接赋予枚举变量，如果要赋值元素，必须要进行强制转换。

```c
enum right {
    True, False
};

int main() {
    enum right a, b, c;
    a=True;
    b=False;
    c=(enum right) 0;
    printf("%d\n", c);
    printf("%d %d", a, b);
    return 0;
}
```
