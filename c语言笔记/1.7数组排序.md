# 选择法

解题思路：以第一个元素与其他元素相比较，若其他元素比他小，则进行交换，若大不进行操作。共进行n-1轮交换即可

```c
#include <stdio.h>

int main() {
    int a[] = {1, 3, -1, 9, 2, 4, 6};
    int a_len= sizeof(a)/sizeof(a[0]);
    int i=0;
    int temp;
  	int min_index;
    for (; i<a_len; i++) {
        min_index = i;
        for (int j=i+1; j<a_len; j++) {
            if (a[i] > a[j]) {
                min_index=j;
            }
        }
        temp=a[i];
        a[i]=a[j];
        a[j]=temp;
    }
    for (int s=0; s<a_len; s++) {
        printf("%d\t", a[s]);
    }
    
}
```



## 冒泡法

解题思路：将相邻两个数进行比较，将小的调到前面。（1)第一次从第一个数开始相互比较到最后一个数，则小的数已经上浮，最大的数已经下沉。第二次从第一个数到倒数第二个数进行比较，小的继续上浮，倒数第二大的数下沉..。共遍历n-1次

```c
#include <stdio.h>

int main() {
    int a[] = {1, 3, -1, 9, 2, 4, 6};
    int a_len= sizeof(a)/sizeof(a[0]);
    int i=0;
    int temp;
    int n=a_len;
    int change=1;
    while (n>0 && change==1) {
        for (; i<n; i++) {
            if (a[i] > a[i+1]) {
                temp = a[i];
                a[i] = a[i+1];
                a[i+1] = temp;
                // 表明发生了交换，有交换证明还可以继续排序
                change=1;
            }
        }
        n--;
    }
    for (int s=0; s<a_len; s++) {
        printf("%d\t", a[s]);
    }
    
}

// 方式2
```

## 遍历法

解题思路：从第一个元素开始查找找出最小的一个作为第一个元素，第二次从第二个元素开始查找第二小的元素。

感觉和选择法差不多

```c
#include <stdio.h>

int main() {
    int a[] = {1, 3, -1, 9, 2, 4, 6};
    int a_len= sizeof(a)/sizeof(a[0]);
    int i=0;
    int temp;
    for (; i<a_len; i++) {
        for (int j=i+1; j<a_len; j++) {
            if (a[i] > a[j]) {
                temp=a[i];
                a[i]=a[j];
                a[j]=temp;
            }
        }
    }
    for (int s=0; s<a_len; s++) {
        printf("%d\t", a[s]);
    }
    
}
```
