# 字符串处理函数

包含在头文件<string.h>中

```c
gets()  //用于输入字符串，返回值是用于存放字符数组的首地址。
puts()  //用于输出字符串，可包含转义字符，因为会对他进行转义
strlen() //测试字符串的长度，不包括'\0'，也可以测试字符数组的长度，实际长度，不含'\0'
strcmp(s1, s2) //字符串比较函数，字符串不能使用==进行比较，这是,返回结果是0，正数，负数，
strcpy(new,origin) //字符串复制函数，C语言中不允许直接将一个字符串赋值给一个字符数组。复制的时候连同'\0'一起复制
strncpy(new, origin, n) //同strcpy，只是n表示复制n个字符
strcat(s1,s2) //用于连接两个以'\0'结尾的字符串，结果会放在s1中，所以s1的长度要足够长
```
