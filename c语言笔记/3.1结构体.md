构造类型数据：结构型，共用型，数组，数组和他们两个的区别是数组存储的元素类型是一样的，另外两个可以存储不同类型的数据。

# 结构体

结构体定义：是由一批数据组合而成的一种新的数据类型。组成结构型数据的每个数据称为结构型数据的**“成员”**，这些成员可以是不同的数据类型。结构体是由用户自己定义

使用结构体数据之前，要自己先定义，和使用函数一个感觉。

定义结构体时并不会分配内存，只有在使用的该结构体的时候才会分配内存。

```c
//格式：
struct 结构体名
{
  数据类型1	成员名1;
  数据类型2	成员名2;
  数据类型3	成员名3;
  ....
};

//结构体中可以嵌套结构体,称为嵌套结构体

```

==结构体可以嵌套本身==

注意：在嵌套本身的时候，成员必须是本结构体的指针型变量，或者结构体指针型数组，不能是结构体变量或结构体数组

```c
struct st {
  int n;
  struct st *s2;
  struct st *s3[10];  // 结构体数组
  
  // 下面是错误示范
  struct st s4;
  struct st s5[10];
}
```



## 结构体变量的初始化

三种方式对结构结构体变量定义：

==第一种：先定义结构体，再定义结构体变量==

格式如下：

```c
struct 结构体名
{
  数据类型1	成员名1;
  数据类型2	成员名2;
  数据类型3	成员名3;
  ....
};
...
struct 结构体名 结构体变量名=初值;

int main() {
    struct birth {
        int year;
        int month;
        int day;
    };
    struct stu {
        int no;
        char name[20];
        struct birth birthday;
    };
    struct birth xiaoming_birth;
    struct stu xiaoming;
  // 下面这种直接赋值的方式是错误的
    xiaoming_birth={2021, 10, 1};
    xiaoming={1, "tom", xiaoming_birth};
  
    struct birth tom_birth = {2021, 10, 1};
    struct stu tom = {1, "tom", tom_birth};
    printf("%d", tom.birthday.year);
    
    return 0;
}
```

==第二种定义结构体的同时定义变量==

```c
// 格式
struct 结构体名
{
  数据类型1	成员名1;
  数据类型2	成员名2;
  数据类型3	成员名3;
  ....
}结构体变量名=初值;

struct stu {
        int no;
        char name[20];
        struct birth birthday;
    }xh={2, "xiaohong", {2021,10,2}},
    xz={3, "xiaozhang", {2021,10,3}};
```



==第三种：直接定义结构体类型变量==

```c
// 注意这的区别，没有结构体类型的说明了
// 此种定义就不能再定义其他的结构体变量了，已经写死为birthday
struct {
        int year;
        int month;
        int day;
    }birthday={2021, 10, 1};
// 
```



注意：

1. 不能直接将一个一坨值赋值给一个结构体变量，要按属性逐个赋值。

2. 但是可以将具有相同结构的结构体变量赋值给另外一个

   ```c
   //比如上面定义的xiaoming_birth和xiaoming
   xiaoming=tom;
   ```
