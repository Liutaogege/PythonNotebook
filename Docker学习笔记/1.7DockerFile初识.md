# DockerFile

dockerfile是用来构建docker镜像的文件！命令参数脚本

构建步骤：

1.编写一个DockerFile文件

2.docker build构建成一个镜像

3.docker run 运行镜像

4.docker push 发布镜像（DockerHUb，阿里云镜像库）

### DockerFile构建过程

**基础知识**
1.每个保留关键字（指令）都必须是大写字母

2.执行顺序从上到下

3.# 表示注释

4.每一个指令都会创建并提交一个新的镜像层
