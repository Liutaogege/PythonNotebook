### 获取施工通告信息

```apl
@api {post} /external_use/construction_dispatch/get_work_information

@apiParamExample {Array} 请求参数
@apiParam     {Dict}       params															参数
@apiParam     {Str}        params.work_date			工作日期，默认今天
@apiParam     {Str}        params.work_category		工作类别，默认全部(不含B3,C2,其他),如果查询某几类，形如work_category=A3或work_category=A1,A2
@apiParam     {Str}        params.work_type			工作种类，默认全部,可选参数为周计划，日计划，月计划，临时计划，若查询某一类，直接传入名称即可
@apiParam     {Str}        params.union_unit		配合单位名称，默认不限制
@apiParam     {Str}        params.work_content		工作内容，默认不限制
@apiParam     {Int}        params.line_no			线别，默认是9
@apiParam     {Str}        params.work_unit		    工作单位，默认不限制								
@apiParam     {Str}        params.principal		    施工负责人名字,默认不限制							
@apiParam     {Int}        params.offset            需要跳过的记录数，默认是0
@apiParam     {Int}        params.limit             返回的记录数，默认是全部
注：以上参数都可以不传

@apiExample {Array} 参数示例
{
    "params": {
        "work_category": "B1",
        "line_no": 9,
        "work_date": "2021-07-05",
    }
}                                                                 
                                                                
                                                            
@apiSuccessExample {Array} 返回数据
@apiSuccess     {List}       data					返回数据
@apiSuccess     {Str}        data.work_code			   作业代码
@apiSuccess     {Str}        data.work_time			   作业时间
@apiSuccess     {Str}        data.work_unit			   施工单位
@apiSuccess     {Str}        data.work_content		   工作内容
@apiSuccess     {Str}        data.work_area		       施工区域
@apiSuccess     {List}       data.work_area_list	   施工区域中包含的场段区域
@apiSuccess     {Str}        data.park_affect_area	   场段影响区域
@apiSuccess     {List}       data.park_affect_area_list	   影响区域中包含的场段区域列表
@apiSuccess     {Str}        data.electric_situation   接触网供电情况
@apiSuccess     {Str}        data.protect_measures     防护措施
@apiSuccess     {Str}        data.cooperate_content    配合内容
@apiSuccess     {Str}        data.principal            施工负责人
@apiSuccess     {Str}        data.principal_tel		   施工负责人联系电话
@apiSuccess     {Str}        data.ask_addr			   请点地址
@apiSuccess     {Str}        data.remark			   备注
注：当查询出错时，code为0，message提示报错信息,data为[]

返回值类型: JSON
```
