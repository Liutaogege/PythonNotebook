# 内部类

内部类就是在一个类的内部再定义一个类，比如A在B内部，B就是内部内，A称为B的外部类

分类：

* 1.成员内部类
* 2.静态内部类
* 3.局部内部类
* 4.匿名内部类

一个类文件中只能有一个public class,但可以有多个class

实例：

```java
package inner_class;

import com.sun.xml.internal.ws.api.model.wsdl.WSDLOutput;

public class Main {
    public static void main(String[] args) {
        // 匿名类，没有将实例保存到变量中
        new A().in();
        new Service(){
            public void TestService() {
                System.out.println("hahaha");
            }
        };
    }

    int age=18;
    static String className = "Main";

    // 内部成员类
    class InnerOne {
        public void in() {
            System.out.println("I am inner" + age);
        }
    }

    // 静态成员类
    static class Innertwo {
        public void in() {
//            System.out.println(age);//不能调用非static属性
            System.out.println(className);
        }
    }

    // 局部成员类
    public void CreateInner() {
        class InnerThird {
            public void in() {
                System.out.println("haha");
            }
        }
    }



}

// 一个类文件中只能有一个public class，但可以有多个class
class A {
    public void in() {
        System.out.println("我将称为匿名类");
    }
}

interface Service {
    void TestService();
}
```
