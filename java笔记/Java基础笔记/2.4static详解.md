static修饰的表示它就是类方法，或者类属性，可以直接由类名进行调用。



两种代码块：

匿名代码块：直接由大括号包起来的

静态代码块：static修饰的大括号，但只执行一次

它们和构造方法的执行顺序是：静态代码块，匿名代码块，构造方法

```java
public class Student {
  // 匿名代码块
  {
    System.out.println("匿名代码块");
  }
  
  // 静态代码块
  static {
    System.out.println("静态代码块");
  }
  
  public Student() {
    System.out.println("构造方法");
  }
}
```
