# 封装

和python中的封装差不多，讲一点不一样的地方，==重载==。

java中的重载是一个类中可以有多个同名的方法，但是传递的参数不同。如下：

```java
public void setInfo(String name) {
        this.name = name;
    }

public void setInfo(String name, int age) {
  this.name = name;
  this.age = age;
}
// 传递不同的参数表示重载，c语言也是这样
```

但是python不是这样：

```python
# python中是不能有同名的方法的，重载的实现就是传递默认值
def setInfo(self, name, age=18):
  self.name = name
  self.age = age
```
