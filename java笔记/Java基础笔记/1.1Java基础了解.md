#  java 

### 不同含义

JDK: Java Development Kit,Java开发者工具，能对Java程序进行开发，包含JRE

JRE: Java Runtime Environment，Java运行时工具，能运行Java程序，但是不能够进行开发

JVM: JAVA Virtual Machine，Java虚拟机，是Java能够跨平台运行的核心

lib：库文件，不常用到

### 注释

* 单行注释：//

* 多行注释: /**/

  ```java
  /*
   多行注释
   多行注释
   */
  ```

* 注释文档: /**+回车

  ```java
  /**
    *  @Author:
    *  @Description:
    *  @Params:
    */
  ```
